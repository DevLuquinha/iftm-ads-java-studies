# ☕ Java Object-Oriented Programming Studies

This repository is dedicated to my studies and practical exercises in **Object-Oriented Programming (OOP)** using **Java**, as part of the **Computer Systems Analysis and Development** program at the **Federal Institute of Triângulo Mineiro (IFTM)**.

The main goal is to consolidate OOP concepts through hands-on exercises, small projects, and academic assignments, while maintaining a clean and well-documented codebase for learning and portfolio purposes.

---

## 📂 Repository Structure

The `main` branch contains the **current semester's work in progress**, focusing on **Object-Oriented Programming in Java**.

Each folder groups exercises or small projects related to a specific OOP concept, making it easy to navigate and review the content.

Completed semesters will be preserved in dedicated branches for historical reference.
